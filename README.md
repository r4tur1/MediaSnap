# MediaSnap - Frontend

MediaSnap is a modern, responsive web application for previewing and downloading media from various platforms like YouTube, Instagram, and Facebook.

## Features
- **Instant Preview**: Paste a link to see video metadata, thumbnails, and available formats.
- **Multiple Formats**: Choose from various video resolutions and audio-only options.
- **Download History**: Keep track of your recent downloads locally.
- **Batch Mode**: Fetch information for multiple links at once.
- **Dark Mode**: Fully responsive UI with a sleek dark theme.

## Tech Stack
- **HTML5 / CSS3**: Custom styles without external frameworks.
- **Vanilla JavaScript**: Pure JS for logic and API interactions.
- **GitHub Pages**: Hosted as a static site.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Configure `API_BASE_URL` in `js/api.js` to point to your backend.

## Deployment
This frontend is designed to be hosted on GitHub Pages.
1. Go to Repository Settings > Pages.
2. Select the `main` branch and `/root` folder.
3. Save and wait for the live link.
